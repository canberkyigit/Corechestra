# Corechestra

A project and work tracking app built with React.

## Features

- Project and task management
- Markdown editor support
- Redux Toolkit for state
- Tailwind CSS for styling

## Tech Stack

- **React** (Create React App)
- **Redux Toolkit** & **React Redux**
- **@uiw/react-md-editor** for markdown
- **Tailwind CSS**
- **Testing Library** for tests

## Setup

```bash
git clone https://github.com/canberkyigit/Corechestra.git
cd Corechestra
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Scripts

| Command          | Description                    |
| ---------------- | ------------------------------ |
| `npm start`      | Run dev server                 |
| `npm run build`  | Production build               |
| `npm test`       | Run tests                      |
| `npm run eject`  | Eject CRA (one-way, optional)   |
